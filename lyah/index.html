debaze / lyah

Lyah is a C++ SIMD mathematics library for computer graphics. It features 2D-4D vectors, 2x2-4x4 square matrices and quaternions, along with common mathematical functions. It is header-only, fully tested and can be easily integrated into other projects. View source repository
